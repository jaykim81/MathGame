# 🍌 바나나 수학 격투! (Math Fighter) 🥊

초등학생(1~3학년) 친구들을 위한 재미있는 수학 학습 격투 게임입니다!
문제를 풀며 귀여운 바나나 캐릭터와 함께 적을 물리치고 수학 왕이 되어 보세요!

## 🎮 주요 기능
- **다양한 난이도**: 
  - **1학년**: 쉬운 덧셈과 뺄셈
  - **1.5학년**: 구구단 곱셈 (학습용)
  - **2학년**: 두 자리 수 연산
  - **3학년**: 복잡한 세 자리 수 연산
- **10단계 스테이지 시스템**: 단계가 올라갈수록 적이 강해지고 제한 시간이 줄어듭니다 (20초 -> 11초).
- **실시간 스코어링**: 정답을 빨리 맞힐수록 더 높은 점수를 얻습니다.
- **캐릭터 커스텀**: '상대방 얼굴 사진'을 업로드하여 나만의 라이벌과 대결할 수 있습니다.
- **명예의 전당**: 로컬 랭킹 시스템을 통해 나의 최고 기록을 등록하고 확인하세요.
- **전투 연출**: 타격 애니메이션, 화면 흔들림, Web Audio API 기반의 실감나는 효과음.

## 🕹️ 조작 방법
- **마우스**: 정답 버튼을 직접 클릭
- **키보드**: 숫자키 `1`, `2`, `3`, `4`를 사용하여 정답 선택

## 🛠️ 기술 스택
- **언어**: HTML5, CSS3, JavaScript (Vanilla JS)
- **사운드**: Web Audio API (합성음)
- **저장**: LocalStorage (랭킹 데이터)

## 🚀 실행 방법
이 저장소를 클론하거나 다운로드한 후, 브라우저에서 `index.html` 파일을 열기만 하면 됩니다!

```bash
# 로컬 서버로 실행 시 (Node.js 설치된 경우)
npx serve -p 8050
```
